# Problem Explorer

An interactive navigator for 37 sustainability & social innovation problem examples — each a real-world case study of how well-intentioned solutions create downstream unsustainabilities.

Companion to [sustainability-and-social-innovation.com/balance](https://sustainability-and-social-innovation.com/balance/).

## Features

- 37 problem examples, color-coded across 5 categories
- Full-text search + category filter
- Deep-linkable cards (URL hash)
- Keyboard nav (← → Esc /)
- Pure HTML/CSS/JS — no build step

## Run locally

Open `index.html` in a browser, or:

```
python3 -m http.server 8765
```

## Embed

Once hosted on GitHub Pages, embed anywhere iframes are allowed:

```html
<iframe src="https://YOUR-USER.github.io/problem-explorer/" width="100%" height="800" frameborder="0"></iframe>
```
